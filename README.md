# gcp-linebot-sample
```shell
npm i
npm run start
```
## Overview
Google Cloud Platform(GCP) の Cloud Functions上でLINEbotを動かす
## Usage
1. Clone this repository.
2. `npm i` to install dependencies.
3. Create .env.local.
```
touch .env.local
echo "LINE_ACCESS_TOKEN=[token]"
```
4. Start development server with `npm run start`
